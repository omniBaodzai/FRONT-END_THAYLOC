I. Front-end
1. html: xây dựng bố cục (cái khung) của trang web

2. css: làm đẹp cho trang web 

3. javascipt: tương tác với người dùng

4. jquery: thư viện (được viết bằng js) giúp viết code js nhanh hơn

5. responsive: tương thích với các thiết bị khác

6. bootstrap: giúp làm responsive nhanh

7. ajax: là công nghệ sử dụng js để cập nhật các
phần tử mà không cần load lại trang web, không seo


8. PSD to HTML/CSS: kiến trúc sư thiết kế bản vẽ, bộ phận
thi công chuyển từ thiết kế sang trang web cụ thể

9. ReactJS: thư viện mạnh hơn jquery
